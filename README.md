# navarro-post1-u3

Estudiante: Yorman Navarro

Descripción del Proyecto: este proyecto consiste en la construcción de una página web de perfil personal funcional y accesible. El objetivo principal es aplicar de forma integrada conceptos avanzados de CSS3, tales como: Estructura Semántica: Uso correcto de etiquetas HTML5 (header, main, section, form).Box Model: Implementación de box-sizing: border-box para un control preciso del tamaño de los elementos.Posicionamiento: Uso de esquemas fixed para el encabezado, relative para contenedores y absolute para elementos superpuestos como el badge de perfil.Formularios Accesibles: Diseño de interfaces interactivas con estados de foco (:focus) y anillos de accesibilidad.

Instrucciones para Ejecución Local: Para visualizar este proyecto en su computadora, siga estos pasos:Clonar o descargar este repositorio en su máquina local.Abrir la carpeta del proyecto en Visual Studio Code.Asegurarse de tener instalada la extensión Live Server.Haga clic derecho sobre el archivo index.html y seleccione la opción "Open with Live Server".La página se abrirá automáticamente en su navegador predeterminado (preferiblemente Google Chrome) en la dirección http://127.0.0.1:5500.
